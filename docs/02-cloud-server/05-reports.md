# گزارش مصرف
---

در بخش `گزارش مصرف` [سرور ابری](https://chabokan.net/products/cloudserver/)، شما می‌توانید به راحتی مصرف آنی و لحظه‌ای منابع سرور خود را بررسی کنید. این اطلاعات شامل میزان استفاده از CPU، RAM، Disk و پنهای باند می‌باشد که به شما کمک می‌کند تا عملکرد سرور خود را بررسی کنید.

![](https://s1.chabokan.net/docs/images/cloud-server-reports.png)

با استفاده از این گزارش، می‌توانید الگوهای مصرف منابع را شناسایی کرده و در صورت نیاز، اقدام به ارتقای منابع سرور خود نمایید. این امکان به شما این اجازه را می‌دهد که با اطمینان بیشتری به رشد و توسعه کسب‌وکار خود ادامه دهید و از بروز مشکلات ناشی از کمبود منابع جلوگیری کنید. برای اطلاع از نحوه ارتقای منابع سرور ابری به این [مستند](/cloud-server/resize/) مراجعه کنید.
