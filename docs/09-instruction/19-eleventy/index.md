# سرویس Eleventy

`Eleventy` یک فریمورک ساده و سریع برای تولید سایت‌های استاتیک است که بر پایه‌ی **Static Site Generation (SSG)** طراحی شده
است.
این فریمورک از جاوااسکریپت و نوارهای مختلف برای ساخت صفحات استاتیک استفاده می‌کند و به کاربران این امکان را می‌دهد که
وب‌سایت‌هایی با سرعت بارگذاری بالا و عملکرد بهینه بسازند. برخلاف روش‌های معمول که داده‌ها در زمان اجرا پردازش می‌شوند،
`Eleventy` تمامی داده‌ها را در زمان کامپایل پردازش می‌کند و به‌صورت استاتیک تولید می‌کند.

یکی از ویژگی‌های برجسته‌ی `Eleventy`، سادگی در استفاده از آن است. کاربران می‌توانند از قالب‌ها و متغیرهای مختلف برای ساخت
صفحات مختلف استفاده کنند و هیچ پیچیدگی اضافی در کارکرد فریمورک مشاهده نمی‌کنند. `Eleventy` به راحتی قابل گسترش است و به
کاربران این امکان را می‌دهد که از پلاگین‌ها و ابزارهای مختلف برای توسعه پروژه‌های خود استفاده کنند.

`Eleventy` از سازگاری کامل با زبان‌های مختلف قالب‌نویسی مانند Nunjucks، Liquid، و Markdown پشتیبانی می‌کند و به شما این
امکان را می‌دهد که پروژه‌های خود را با هر زبانی که ترجیح می‌دهید، پیاده‌سازی کنید.
