# سرویس Deno

`Deno` یک محیط اجرای مدرن برای جاوااسکریپت و تایپ‌اسکریپت است که توسط Ryan Dahl ساخته شده و با هدف رفع محدودیت‌های Node.js توسعه یافته است. این پلتفرم بدون نیاز به پکیج‌منیجر، از ماژول‌های مبتنی بر URL پشتیبانی می‌کند و ابزارهای داخلی مانند تست و فرمت‌کننده کد را ارائه می‌دهد. همچنین امنیت به‌صورت پیش‌فرض فعال است و دسترسی‌ها فقط با اجازه صریح امکان‌پذیر است.
