# سرویس Nginx

`Nginx` یک وب‌سرور و پروکسی معکوس قدرتمند و متن‌باز است که توسط **Igor Sysoev** توسعه داده شده و با هدف ارائه کارایی
بالا، مصرف کم منابع و مدیریت هم‌زمان تعداد زیادی اتصال طراحی شده است. این نرم‌افزار از معماری رویدادمحور و غیرهمزمان
استفاده می‌کند که آن را برای ترافیک‌های سنگین و مقیاس‌پذیر بسیار مناسب می‌سازد. Nginx علاوه بر وب‌سرور، به‌عنوان **Load
Balancer**، **Reverse Proxy** و **HTTP Cache** نیز به‌کار می‌رود و با پیکربندی منعطف، امنیت بالا و پشتیبانی از
پروتکل‌هایی مانند HTTP/2 و HTTPS، یکی از محبوب‌ترین انتخاب‌ها در زیرساخت‌های مدرن وب محسوب می‌شود.
