# سرویس Nitro

`Nitro` یک موتور سرور سبک و مدرن برای اجرای منطق سمت سرور در اپلیکیشن‌های وب است که با تمرکز بر کارایی بالا و حداقل
سربار طراحی شده است. این ابزار که هسته‌ی سرور Nuxt 3 را تشکیل می‌دهد، امکان پیاده‌سازی API، رندر سمت سرور و منطق
backend-for-frontend را به شکلی ساده و بهینه فراهم می‌کند و با پشتیبانی از محیط‌های مختلف مانند Node، Serverless و Edge،
استقرار سریع و انعطاف‌پذیر را ممکن می‌سازد. Nitro گزینه‌ای مناسب برای پروژه‌هایی است که سرعت، سادگی و هماهنگی نزدیک با
فرانت‌اند برایشان اهمیت دارد.
