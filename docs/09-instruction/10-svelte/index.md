# سرویس Svelte

`Svelte` یک فریم‌ورک مدرن برای ساخت رابط‌های کاربری در جاوااسکریپت و تایپ‌اسکریپت است که با رویکرد متفاوت نسبت به سایر
فریم‌ورک‌ها توسعه یافته است. برخلاف React یا Vue که بیشتر در زمان اجرا کار می‌کنند، `Svelte` کد شما را در زمان کامپایل به
جاوااسکریپت بهینه‌شده تبدیل می‌کند، و باعث افزایش سرعت و کاهش حجم نهایی برنامه می‌شود. این فریم‌ورک سبک، ساده و مبتنی بر
کامپوننت است و تجربه توسعه روان و خوانایی کد بالایی را ارائه می‌دهد.
