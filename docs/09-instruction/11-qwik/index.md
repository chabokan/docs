# سرویس Qwik

`Qwik` یک فریم‌ورک مدرن و نوآورانه برای ساخت رابط‌های کاربری وب است که با تمرکز ویژه بر **سرعت بارگذاری اولیه** و
**بهینه‌سازی عملکرد** طراحی شده است. ایده‌ی اصلی Qwik بر پایه‌ی *Resumability* شکل گرفته؛ به این معنا که برنامه بدون
نیاز به اجرای کامل جاوااسکریپت در مرورگر، به‌صورت آنی قابل استفاده است.

برخلاف بسیاری از فریم‌ورک‌های رایج که نیاز به Hydration کامل دارند، `Qwik` فقط در زمان نیاز و به‌صورت تدریجی
جاوااسکریپت را بارگذاری می‌کند. این رویکرد باعث کاهش چشمگیر حجم باندل اولیه و بهبود معیارهای Core Web Vitals می‌شود.
Qwik گزینه‌ای بسیار مناسب برای وب‌سایت‌ها و اپلیکیشن‌هایی است که **SEO، Performance و تجربه کاربری سریع** در آن‌ها
اهمیت بالایی دارد.
