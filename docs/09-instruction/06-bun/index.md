# سرویس Bun

`Bun` یک محیط اجرای مدرن و بسیار سریع برای جاوااسکریپت و تایپ‌اسکریپت است که توسط **Jarred Sumner** توسعه داده شده و با
تمرکز بر **کارایی بالا و یکپارچگی ابزارها** ساخته شده است. Bun از موتور **JavaScriptCore** استفاده می‌کند و به‌صورت
پیش‌فرض یک **پکیج‌منیجر سازگار با npm**، باندلر و تست‌رانر داخلی ارائه می‌دهد. این پلتفرم با هدف جایگزینی ابزارهای
پراکنده‌ای مانند Node.js، npm، webpack و jest در یک ابزار واحد طراحی شده و سرعت اجرای بسیار بالاتری، به‌ویژه در نصب
پکیج‌ها و اجرای اسکریپت‌ها، فراهم می‌کند.
