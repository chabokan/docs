# روش های اتصال به ArangoDB
---

![اتصال به سرویس دیتابیس ArangoDB چابکان](https://s1.chabokan.net/docs/images/arangodb4.jpg)

```properties
Username : نام کاربری
Password : رمز عبور
```

بعد از طی کردن مراحل بالا وارد محیط ArangoDB می شوید.
