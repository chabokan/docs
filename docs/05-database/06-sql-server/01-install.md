# نصب و راه اندازی SQL Server
---

در این بخش می خواهیم آموزش نحوه نصب و راه اندازی دیتابیس SQL Server را در سرویس [دیتابیس اس کیو ال سرور](https://chabokan.net/services/mssql/) به شما آموزش دهیم.
همانطور که در گیف زیر مشاهده میکنید، ابتدا سرویس `MSSQL` را از طریق قسمت ساخت سرویس جدید > دیتابیس ابری > `MSSQL` می سازیم.

![نصب و راه‌اندازی دیتابیس SQL Server در چابکان](https://s1.chabokan.net/docs/gifs/mssql-install.gif)

سرویس های ابری چابکان امکان راه اندازی با نسخه های مختلف را نیز دارند که شما می توانید نسخه موردنظر خود را انتخاب کنید.

![نصب و راه‌اندازی دیتابیس SQL Server در چابکان](https://s1.chabokan.net/docs/images/sqlserver-start.png)
