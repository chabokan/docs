:::tip نکته
در تمام برنامه ها تنظیمات `NGINX` نیاز نیست و چابکان به صورت خودکار این کار را برای شما انجام می دهد و اگر چیزی
غیر از این تنظیمات پیش‌فرض نیاز دارید باید آن را تغییر دهید.

```nginx title="تنظیمات پیش‌فرض NGINX توسط چابکان:"
server {
        listen 80 default_server;
        listen [::]:80 default_server;
        root /app/dist;
        index index.html index.htm;
        server_name _;

        location / {
                try_files $uri $uri/ /index.html =404;
        }
}
```

:::

برای تغییر تنظیمات پیش‌فرض `NGINX` باید یک فایل با نام `nginx.conf` در روت اصلی برنامه خود ایجاد کنید و تنظیمات
مورد نظر خود را درآن قرار دهید. با این کار هربار که برنامه شما ری‌استارت شود تنظیمات که برای `NGINX` در نظر گرفته اید
جایگزین خواهد شد.

پس از ایجاد فایل `nginx.conf` و تنظیم آن می‌بایست سرویس را یکبار ری‌استارت کنید تا تنظیمات اعمال شوند. همچنین برای بررسی
اینکه تنظیمات به درستی اعمال شده اند می‌توانید به لاگ‌های سرویس مراجعه کنید. در صورتی که پیام زیر را در انتهای لاگ‌ها
مشاهده کردید به این معناست که تنظیمات `NGINX` با موفقیت در سرویس شما اعمال شده اند:

```shell
> Copy NGINX Configuration ...
> Starting NGINX Service ...
Starting nginx: nginx.
> Done, Your Service Is Ready.
```
