## سرویس‌های اشتراکی

::::info سرویس‌های قابل استفاده

سرویس‌های اشتراکی قابل ایجاد در هاست **{props.platform}** به صورت زیر می‌باشند:
<ul>
    {
        props.services.map(service => {
            return <li key={service}>{service}</li>
        })
    }
</ul>

:::tip همچنین می‌توانید تفاوت سرویس‌های اشتراکی و اختصاصی را در این [مستند](/general-tips/share-db-vs-dedicated-db/) مشاهده فرمایید.
:::
